# Wi-Fi Analyzer

Wi-Fi signal analyzer and channel scanner.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install wifi-analyzer
```

### Fedora/RPM
```bash
sudo dnf install wifi-analyzer
```
